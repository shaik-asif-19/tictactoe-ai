# Tic Tac Toe Game

A classic Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- **Two-player gameplay** - Player X vs Player O
- **Real-time status** - See whose turn it is
- **Win detection** - Automatically detects wins and draws
- **Responsive design** - Works on all screen sizes
- **Easy reset** - Play multiple games with the reset button

## How to Play

1. Open `index.html` in your browser
2. Players take turns clicking on empty cells
3. Player X goes first
4. First player to get 3 in a row (horizontally, vertically, or diagonally) wins
5. Click "Reset Game" to start a new game

## Game Rules

- Click any empty cell to make your move
- Players alternate between X and O
- Winning combinations:
  - Horizontal rows (top, middle, bottom)
  - Vertical columns (left, center, right)
  - Diagonal lines (both directions)
- If all cells are filled with no winner, the game is a draw

## Files

- `index.html` - Game structure
- `style.css` - Styling and animations
- `script.js` - Game logic and interactivity
